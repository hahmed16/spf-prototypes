ملخص نماذج البيانات والقوالب
الموقع المرجعي: Directorate-WorkInjuries/shared

النماذج المشتركة

1. نموذج المؤمن عليه
يُستخدم أساسًا في:
- بدلات الانقطاع عن العمل
- التظلمات
- طلبات العرض على المؤسسات الصحية المرخصة

الحقول:
- الاسم الكامل
- الرقم المدني
- رقم التأمين
- تاريخ الميلاد
- الجنس
- الجنسية
- الحالة التأمينية
- تاريخ التسجيل
- نوع الاشتراك
- رقم الهاتف
- البريد الإلكتروني
- تاريخ انتهاء البطاقة المدنية عند الحاجة

2. نموذج جهة العمل
يُستخدم أساسًا في:
- بدلات الانقطاع عن العمل
- التظلمات
- طلبات العرض على المؤسسات الصحية المرخصة

الحقول:
- اسم جهة العمل
- السجل التجاري
- رقم المنشأة
- المسمى الوظيفي
- تاريخ الالتحاق
- موقع العمل
- القطاع
- نوع جهة العمل
- بيانات الفرع:
  id
  name
  state
  governorate
- رقم الهاتف

3. نموذج مقدم الطلب
يظهر في أغلب أنواع الطلبات

الحقول:
- الاسم الكامل
- الرقم المدني
- الصفة أو الدور
- رقم الهاتف
- البريد الإلكتروني
- المنطقة عند الحاجة
- الولاية عند الحاجة
- الدولة عند الحاجة
- رقم الموظف عند الحاجة
