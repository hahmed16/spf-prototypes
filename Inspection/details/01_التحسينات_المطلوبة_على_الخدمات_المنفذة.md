# التحسينات المطلوبة على الخدمات المنفذة حالياً

## 1. خدمة الشكاوى والبلاغات

- إضافة مكون توثيق المخاطبات (Shared Component)
- إضافة نظام الخرائط الجغرافية ضمن تفاصيل الزيارة
- إضافة شاشة تحليل جغرافي لرؤساء الأقسام ومدير الدائرة ومحلل العمليات
- إضافة نظام المخالفات والعقوبات مرتبط بنتائج البحث
- تعزيز البيانات من الأنظمة الداخلية والخارجية

### شاشة تحليل جغرافي

- **الأدوار المسموحة**: جميع الأدوار
- **الإجراءات الممكنة**:
  - جميع الأدوار: عرض البيانات الجغرافية، إضافة فلاتر، تصدير التقارير

## 2. خدمة الزيارات الميدانية

- إضافة مكون توثيق المخاطبات (Shared Component)
- إضافة مكون تسجيل المكالمات المرئية مدمج داخل شاشة تفاصيل الزيارة
- إضافة نظام الخرائط الجغرافية ضمن تفاصيل الزيارة
- تعزيز البيانات من الأنظمة الداخلية والخارجية

## 3. خدمة التظلمات

- إضافة مكون توثيق المخاطبات (Shared Component)
- إضافة مكون تسجيل المكالمات المرئية (Shared Component)
- تعزيز البيانات من الأنظمة الداخلية والخارجية

## 4. شاشات البحث والتحليل المتقدمة

- إضافة البحث المتقدم (يمكن تطبيقه لاحقاً)
- الترابط بين شاشات التحليل الموجودة وعدم وجود تعارض

## 5. السجلات والتوثيق

- عرض السجل الزمني لكل طلب أو بلاغ أو زيارة بشكل تفصيلي
- إما ضمن تفاصيل البلاغ أو في شاشة مستقلة

### شاشة السجل الزمني (إذا كانت مستقلة)

- **الأدوار المسموحة**: مدير الدائرة، رئيس قسم المتابعة والبلاغات، رئيس قسم الزيارات الميدانية، موظف المتابعة والبلاغات، موظف الزيارات الميدانية
- **الإجراءات الممكنة**:
  - مدير الدائرة: عرض الكل، تصدير السجلات، إضافة فلاتر مخصصة
  - رئيس قسم المتابعة والبلاغات: عرض الكل، تصدير السجلات، إضافة فلاتر مخصصة
  - رئيس قسم الزيارات الميدانية: عرض الكل، تصدير السجلات، إضافة فلاتر مخصصة
  - موظف المتابعة والبلاغات: عرض المخصصة، تصدير السجلات
  - موظف الزيارات الميدانية: عرض المخصصة، تصدير السجلات

## 6. مركز التقارير

- تعزيز التقارير الخاصة بكل دور فقط
- تعزيز التقارير الحالية لكل دور بما هو ممكن ومطلوب فقط
- عرض مصمم التقارير كفكرة (قابلية التنفيذ قيد الدراسة)

## 7. المكونات المشتركة (Shared Components)

- مكون توثيق المخاطبات

### مكون توثيق المخاطبات

- **الأدوار المسموحة**: جميع الأدوار
- **الإجراءات الممكنة**:
  - جميع الأدوار: عرض التوثيق، إضافة سجلات جديدة، إرفاق مرفقات
