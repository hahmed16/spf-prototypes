# نظام توثيق المخاطبات والاتصالات - مواصفات النظام

## نظرة عامة
آلية توثيق المخاطبات والاتصالات بأطراف الشكوى دون التأثير على حالة الشكوى أو البلاغ - فقط للتوثيق

## المتطلبات الأساسية

### 1. التوثيق المنفصل
- نظام توثيق منفصل عن سير العمل الرئيسي
- عدم تأثير التوثيق على حالة الشكوى
- الحفاظ على حالة الشكوى كما هي
- إمكانية التوثيق في أي مرحلة

### 2. أنواع التواصل
- [ ] التواصل الهاتفي
- [ ] البريد الإلكتروني
- [ ] واتساب

- [ ] اللقاءات المباشرة
- [ ] المراسلات الرسمية
- [ ] الفاكس
- [ ] أخرى

### 3. بيانات التوثيق المطلوبة
- تاريخ ووقت التواصل
- نوع التواصل
- الطرف الآخر (الاسم والصفة)
- وسيلة التواصل
- محتوى التواصل
- ملخص التواصل
- المرفقات (إن وجدت)
- الموظف المسؤول
- النتيجة (إن وجدت)

### 4. الميزات المطلوبة

## سير العمل المقترح

### إضافة مخاطبة جديدة
1. اختيار نوع التواصل
2. إدخال بيانات الطرف الآخر
3. تسجيل محتوى التواصل
4. إرفاق المرفقات (إن وجدت)
5. حفظ التوثيق
6. عدم التأثير على حالة الشكوى

### عرض سجل التواصلات
1. عرض جميع التوثيق
2. الفلترة حسب النوع
3. الفلترة حسب التاريخ
4. البحث في المحتوى
5. عرض التفاصيل الكاملة

## الشاشات المطلوبة

### 1. لوحة توثيق المخاطبات
- زر إضافة مخاطبة جديدة
- عرض سجل التواصلات
- الفلترة والبحث
- التصدير

### 2. شاشة إضافة مخاطبة جديدة
- حقول إدخال البيانات
- اختيار نوع التواصل
- إرفاق المرفقات
- حفظ وإلغاء

### 3. شاشة عرض تفاصيل المخاطبة
- عرض جميع البيانات
- عرض المرفقات
- إمكانية التعديل
- الطباعة

### 4. سجل التواصلات
- جدول شامل
- الفلترة المتقدمة
- البحث
- التصدير

## التكامل مع الأنظمة الأخرى

### التكامل مع الشكاوى
- الربط برقم الشكوى
- عرض التوثيق في تفاصيل الشكوى
- عدم التأثير على سير العمل

### التكامل مع الزيارات
- الربط برقم الزيارة
- توثيق التواصل قبل/بعد الزيارة
- تسجيل الملاحظات

### التكامل مع التظلمات
- الربط برقم التظلم
- توثيق المراجعات
- تسجيل القرارات

## قواعد الأعمال

### قواعد التوثيق
1. يمكن التوثيق في أي مرحلة من مراحل الشكوى
2. لا يؤثر التوثيق على حالة الشكوى
3. يجب توثيق جميع التواصلات الهامة
4. يمكن إرفاق المرفقات
5. يمكن البحث والفلترة

### قواعد الوصول
1. جميع الموظفين المختصين يمكنهم التوثيق
2. يمكن للمشرفين عرض جميع التوثيق
3. يمكن للمديرين الوصول الكامل
4. التوثيق للقراءة فقط بعد الإغلاق

### قواعد الحفظ
1. الحفظ التلقائي بعد الإدخال
2. لا يمكن الحذف بعد الحفظ
3. يمكن التعديل خلال 24 ساعة
4. الأرشفة التلقائية بعد إغلاق الشكوى
