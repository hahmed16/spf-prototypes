# الخدمات المفقودة بالكامل

## 1. طلبات الأمان الوظيفي

### سير العمل الموحد

يتم توجيهها لقسم المتابعة والبلاغات للمتابعة → يمكن إحالتها لقسم التفتيش عند الحاجة → اعتماد المدير النهائي

### الشاشات المطلوبة:

#### قائمة طلبات الأمان الوظيفي

- **الأدوار المسموحة**: مدير الدائرة، رئيس قسم المتابعة والبلاغات، موظف قسم المتابعة والبلاغات، موظف قسم التفتيش، رئيس قسم التفتيش
- **الإجراءات الممكنة**:
  - مدير الدائرة: عرض الكل، الموافقة/الرفض، إحالة للمراجعة، طلب معلومات إضافية
  - رئيس قسم المتابعة والبلاغات: عرض الكل، إحالة للموظفين، طلب معلومات إضافية، إغلاق الطلب
  - موظف قسم المتابعة والبلاغات: عرض المخصصة، إضافة ملاحظات، طلب معلومات إضافية، إحالة للمراجعة
  - موظف قسم التفتيش: عرض المخصصة، إضافة ملاحظات، طلب معلومات إضافية، إحالة للمراجعة
  - رئيس قسم التفتيش: عرض الكل، إحالة للموظفين، طلب معلومات إضافية

#### تفاصيل طلب الأمان الوظيفي

- **الأدوار المسموحة**: مدير الدائرة، رئيس قسم المتابعة والبلاغات، موظف قسم المتابعة والبلاغات، موظف قسم التفتيش، رئيس قسم التفتيش
- **الإجراءات الممكنة**:
  - مدير الدائرة: عرض التفاصيل، الاعتماد النهائي، إحالة للمراجعة
  - رئيس قسم المتابعة والبلاغات: عرض التفاصيل، إحالة للموظفين، طلب معلومات إضافية، إحالة لقسم التفتيش
  - موظف قسم المتابعة والبلاغات: عرض التفاصيل، إضافة ملاحظات، طلب معلومات إضافية، إحالة للمراجعة
  - موظف قسم التفتيش: عرض التفاصيل، إضافة ملاحظات، طلب معلومات إضافية، إحالة للمراجعة
  - رئيس قسم التفتيش: عرض التفاصيل، إحالة للموظفين، طلب معلومات إضافية
- **المحتوى**: لوحة التحقق من الإنهاء، لوحة التحليل المالي
